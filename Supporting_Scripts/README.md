# Supporting SuiteScripts
User Event scripts that power the invoice line grouping feature by aggregating line data and storing it in a custom field for the FreeMarker template to render.

## Scripts
- **invoice_grouping_ue_v1.js** - beforeLoad version (groups on print)
- **invoice_grouping_ue_beforesubmit.js** - beforeSubmit version (groups on save)
- **invoice_grouping_ue_v2.js** - Enhanced beforeLoad with expense category handling
